About

This tool supports .xls, .xlsx, and .csv files and performs logical comparisons by Part, Reference, and Quantity. Unlike text diff tools, BOM Diff understands BOM structure and helps engineers track real changes.

Made by an engineer. Distributed as freeware. No installation. No registration.
