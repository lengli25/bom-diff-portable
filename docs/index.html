About

BOM Diff is a lightweight engineering tool for comparing BOM (Bill of Materials) files.
It supports .xls, .xlsx, and .csv formats, and performs logical comparisons by Part, Reference, and Quantity.

Unlike generic text diff tools, BOM Diff understands the structure of BOMs and helps engineers track real changes with clarity and precision.

Designed by an engineer, for engineers. No installation required.
